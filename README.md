# M4m Sig Serve

## Usage

- clone repo to local
- cd sig-server
- [config env](#env)
- npm install
- num run start
    - server listen at 127.0.0.1:7001

## Env

config env.json at "./config/env.json"

```json
{
  "NODE_URL_MUMBAI": "",
  "GAME_SIGNER_PRIV_KEY": ""
}
```
